# Hostel Room Rent Calculator

A simple Python script to calculate and split the total cost of renting a hostel room and sharing common expenses among a group of people.

## Features
- Calculates total expenses by summing up:
  - Rent amount
  - Food orders
  - Electricity bill
  - Water bill
  - Internet bill
- Automatically splits the total cost equally among the given number of people sharing the room.

## Requirements
- Python 3.x

## How to Use
1. Clone or download this project.
2. Open a terminal or command prompt and navigate to the project directory.
3. Run the script using Python:
   ```bash
   python hostel_rent.py
   ```
4. Enter the required amounts when prompted:
   - Total rent amount
   - Total food ordered
   - Electricity bill
   - Water bill
   - Internet bill
   - Total number of people sharing the rent
5. The script will display the total amount each person needs to pay.

## Example
```
Enter the total rent amount = 5000
Enter the total food ordered = 1200
Enter the electricity bill = 400
Enter the water bill = 100
Enter the amount of internet bill = 300
Enter the total number of people sharing the rent = 2
Total amount per person to pay = 3500.0
```
